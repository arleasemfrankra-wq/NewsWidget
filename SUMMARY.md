## 🐛 已知问题

### 1. 数据源 API 失效
- 知乎、微博等主流平台的 API 需要更新
- 需要研究更稳定的数据获取方式

### 2. 网页抓取挑战
- 很多网站使用 JavaScript 渲染
- 需要使用 Puppeteer 或类似工具
- 或者寻找可靠的第三方 API
