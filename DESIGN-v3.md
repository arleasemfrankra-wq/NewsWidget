### 4. 相对时间显示 ⏰

**更人性化的时间表达**

```javascript
function getRelativeTime(timeStr) {
  const diff = Math.floor((now - time) / 1000);
  
  if (diff < 60) return '刚刚';
  if (diff < 3600) return `${Math.floor(diff / 60)}分钟前`;
  if (diff < 86400) return `${Math.floor(diff / 3600)}小时前`;
  if (diff < 604800) return `${Math.floor(diff / 86400)}天前`;
  
  return timeStr;
}
```

**效果：**
- `刚刚` - 1分钟内
- `5分钟前` - 1小时内
- `2小时前` - 24小时内
- `1天前` - 7天内

## 🎨 设计细节

### 配色方案

**主色调：**
- 背景：`rgba(255, 255, 255, 0.95)` - 更高透明度
- 毛玻璃：`blur(50px)` - 更强模糊
- 饱和度：`200%` - 更鲜艳

**分类色彩：**
- 热点：红色渐变（活力）
- 科技：蓝色渐变（科技感）
- 财经：绿色渐变（增长）
- 西班牙：黄色渐变（阳光）
- 国际：灰色渐变（中性）

### 间距系统

**8px 基准：**
- 4px - 最小间距
- 8px - 小间距
- 12px - 中间距
- 16px - 大间距
- 20px - 超大间距

### 圆角系统

**统一圆角：**
- 6px - 小元素（标签）
- 8px - 中元素（按钮、卡片）
- 10px - 大元素（Toast）
- 12px - 超大元素（容器）

### 字体系统

**大小：**
- 17px - 主标题
- 13px - 新闻标题
- 11px - 来源、按钮
- 10px - 时间、标签

**粗细：**
- 700 - 标题、标签
- 600 - 新闻标题、来源
- 500 - 普通文本
